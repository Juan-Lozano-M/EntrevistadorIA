# InterviewAI — Frontend

React + TypeScript + Vite SPA for the InterviewAI interview-simulation platform.
Consumes the backend REST API (`VITE_API_BASE_URL`, default `http://localhost:8080/api`).

## Scripts
- `npm run dev` — start the dev server
- `npm run build` — type-check and build for production
- `npm run test` — run the Vitest test suite
